# setTarget.sh
Objective: Set alias to Target IP Address in /etc/hosts.

Usage:
	sudo setTarget \<IP Address> \<Alias Name>
	
Steps:
1) Save the setTarget.sh
2) sudo chmod 777 setTarget.sh
3) sudo ln -s <Path of setTarget.sh> /usr/local/bin/setTarget
